<huss-carousel>

A carousel for either images or other elements. If you display other elements, the carousel will split them per slide and adjust the number of elements depending on screen size.

Attributes

NameDescriptionTypeDefault
variantAn option telling the carousel that you are displaying either images or other components.image | anyContent
slideDelayHow long is one slide displayed.number'5500'
autoplayShould the carousel start autoplaying.booleanundefined
elementsInSlideDesktopHow many elements should fit in one slide on large screen.number3
elementsInSlideTabletHow many elements should fit in one slide on medium screen.number2
elementsInSlideMobileHow many elements should fit in one slide on small screen.number1
shouldHugShould the remaining elements on last slide be rendered closely together, rather than laid out with flex.boolean
